---
title: "Spacing"
---

Our spacing tokens follow a base 8 rule. This rule is applied accross our tokens
with the exception of 4px `xxs`. All spacing should follow these tokens.

## Tokens

| Name | value |
|------|-------|
| 0    | 0     |
| auto | auto  |
| xxs  |  4    |
| xs   | 8     |
| sm   | 16    |
| md   | 24    |
| lg   | 32    | 
| lg-mod| 40   |
| xl   |  48   |
| xxl  |  64   |
| xxl-mod | 80 |
