<!-- Description of component -->
<p className="font-xs font-light font--subhead">
Non-actionable UI element that only contains text and does 
not require the user to take action but serves only to 
inform the user before taking action, contextualizing an action or label/icon, but when 
used as a Popover, the element takes on a more persistent appearance and allows the user to take
action on the content inside of the popover.
</p>
