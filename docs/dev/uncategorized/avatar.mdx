# Avatar

<!-- Description of component -->
<p className="font-xs font-light font--subhead">
An avatar is an image that represents a person. At The Washington Post we present avatars as circular photographs which represent individual contributors such as reporters and columnists.

## Avatar Groups
To represent multiple individuals, Medium, Small, and Extra Small avatars can be shown in groupings.
Example of uses are:
- `Medium` - used on Article page bylines
- `Small` - used on Homepage bylines, nav bar
