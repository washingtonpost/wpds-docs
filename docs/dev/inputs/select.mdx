## Requirements
We built Select using react-aria library https://react-spectrum.adobe.com/react-aria/useSelect.html

React Aria requires applications to be wrapped in their SSRProvider. This ensures that auto generated ids are consistent between client and server. https://react-spectrum.adobe.com/react-aria/SSRProvider.html. Here's an example of it. You can read more about their server side rendering implementation here. https://react-spectrum.adobe.com/react-aria/ssr.html

```
import {SSRProvider} from '@react-aria/ssr';

<SSRProvider>
 <YourApp />
</SSRProvider>
```

The following peer deps should be installed:

- [@react-stately/select](https://react-spectrum.adobe.com/react-stately/useSelectState.html)
- [@react-aria/select](https://react-spectrum.adobe.com/react-aria/useSelect.html)
- [@react-aria/listbox](https://react-spectrum.adobe.com/react-aria/useListBox.html)
- [@react-aria/utils](https://react-spectrum.adobe.com/react-aria/mergeProps.html)
- [@react-aria/button](https://react-spectrum.adobe.com/react-aria/useButton.html)
- [@react-aria/interactions](https://react-spectrum.adobe.com/react-aria/useFocus.html)
- [@react-aria/focus](https://react-spectrum.adobe.com/react-aria/FocusRing.html)
- [@react-aria/overlays](https://react-spectrum.adobe.com/react-aria/DismissButton.html)
- [@react-stately/collections](https://react-spectrum.adobe.com/react-aria/useListBox.html)

```sh
npm i @react-stately/select @react-aria/select @react-aria/listbox @react-aria/utils @react-aria/button @react-aria/interactions @react-aria/focus @react-aria/overlays @react-stately/collections
```
