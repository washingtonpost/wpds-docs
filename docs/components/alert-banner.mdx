## Options

### Variants

4 predefined alert banners each tied to our symantic messaging on our site. They are Warning, Information, Success, and Error.

```jsx
// preview

import { AlertBanner } from "@washingtonpost/wpds-ui-kit";

export default function Example() {
  return (
    <AlertBanner.Root variant="error">
      <AlertBanner.Content>Hey There</AlertBanner.Content>
    </AlertBanner.Root>
  );
}
```
