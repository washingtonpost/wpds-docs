## Usage guidelines

<!-- Start of a guide -->
<div className="mt-lg mb-sm">

### Improper aspect ratio
The image in the avatar needs a 1:1 aspect ratio to round out properly. When using the wrong 
aspect ratio, the avatar shape comes out wrong.

</div>
<Guide guidance="don't">
  <Avatar size="large" imgUrl="/img/design/avatar/badprofile.png"/>
</Guide>



<div className="mt-lg mb-sm">

### Must be a single profile
The avatar should not be used to display a photo that has more than one subject. 

</div>
<Guide guidance="don't">
  <Avatar size="large" imgUrl="/img/design/avatar/squarebad.png"/>
</Guide>
