## Usage guidance
<!-- Start of a guide -->
<br/>

### Avoid small widths
If a small width text input is required, be sure the expected input will be short.  

<Guide guidance="caution">
  <div style={{width:100}}>
<Input
  labelText="State"
  name="State"
  onBlur={function noRefCheck(){}}
  onFocus={function noRefCheck(){}}
  size="default"
  type="text"
  value="MD"
/>
</div>

</Guide>

<br/>

### Vary input heights
If more than one input is being used, all inputs must be the same size variant.   

<Guide guidance="don't">
      <div className="mr-sm" style={{width:150}}>
<Input
  labelText="City"
  name="City"
  onBlur={function noRefCheck(){}}
  onFocus={function noRefCheck(){}}
  size="small"
  type="text"
/>
</div>
  <div style={{width:100}}>
<Input
  labelText="State"
  name="State"
  onBlur={function noRefCheck(){}}
  onFocus={function noRefCheck(){}}
  size="default"
  type="text"
  value="MD"
/>
</div>

</Guide>
